# Fair Topic Allotment System

## Project Overview

The Fair Topic Allotment System is a web-based application designed to allocate project topics to students in a fair and transparent manner. Students can submit their topic preferences, and the system automatically assigns topics based on availability and priority.

This system helps avoid manual conflicts and ensures an efficient and unbiased topic allocation process for academic project fairs.

---

## Features

* Student registration and login
* Topic preference submission
* Admin panel for managing topics
* Automatic topic allocation
* Real-time seat availability
* Transparent allocation process
* Secure authentication system

## Installation and Setup

### 1. Clone the repository

```
git clone https://github.com/chandu-bammidi/fair-topic-allot.git
```

### 2. Navigate to project folder

```
cd fair-topic-allot
```

### 3. Install backend dependencies

```
cd server
npm install
```

### 4. Install frontend dependencies

```
cd ../client
npm install
```

### 5. Start the backend server

```
cd ../server
npm start
```

### 6. Start the frontend

```
cd ../client
npm start
```
